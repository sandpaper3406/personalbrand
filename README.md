# James Markley Personal Brand Website

A fast, modern personal brand website for jamesmarkley.com. The implementation is a static frontend with reusable JavaScript components, premium dark-mode styling, responsive sections, smooth transitions, and placeholder integrations for scheduling, venture links, social links, and headshot/logo assets.

## Local Preview

```bash
node server.js
```

Then open `http://localhost:4173`.

## Structure

- `index.html` contains the SEO shell and app mount.
- `src/main.js` contains section data and reusable component renderers.
- `src/styles.css` contains the responsive visual system, layout, animation, and interaction polish.
- `server.js` provides a tiny local static server without requiring npm.
